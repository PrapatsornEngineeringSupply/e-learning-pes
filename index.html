🚀 วิธีติดตั้ง LearnKPI v2
ขั้นตอนที่ 1: สร้าง Google Sheet
ไปที่ sheets.new สร้าง Sheet ใหม่ ตั้งชื่อ เช่น "LearnKPI Database"
คัดลอก Sheet ID จาก URL: https://docs.google.com/spreadsheets/d/SHEET_ID/edit
ขั้นตอนที่ 2: สร้าง Google Drive Folder สำหรับ MP4
ไปที่ Google Drive → New → Folder → ตั้งชื่อ "LearnKPI_Videos"
เปิด Folder → คัดลอก Folder ID จาก URL: https://drive.google.com/drive/folders/FOLDER_ID
ขั้นตอนที่ 3: ติดตั้ง Apps Script
ใน Google Sheet → Extensions → Apps Script
วางเนื้อหาจากไฟล์ Code.gs แทนที่ของเดิม
แก้บรรทัดที่ 14–15:
const SHEET_ID = 'ใส่ Sheet ID ที่นี่';
const UPLOAD_FOLDER_ID = 'ใส่ Folder ID ที่นี่';
ขั้นตอนที่ 4: Deploy เป็น Web App
Deploy → New Deployment → Web app
• Execute as: Me
• Who has access: Anyone
→ คัดลอก URL ที่ได้
ขั้นตอนที่ 5: Initialize & Login
วาง URL ในช่อง Script URL → คลิก Initialize → Login ด้วย admin / admin1234
